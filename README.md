Requirements
============

Python, pip, mysql


Installation
============

    $ pip install -r requirements.txt


Usage
======

Copy config.yaml.sample to config.yaml. Fill in relevant values.

    $ cp config.yaml.sample config.yaml

Make sure the bash start/stop scripts are executable.

    $ chmod +x *.sh

Start the bot
==============

    $ ./start.sh

Stop the bot
=============

    $ ./stop.sh
